# 📄 Mi Portafolio Minimalista

¡Bienvenido a mi portafolio en la web! 🌐✨

Este proyecto nace gracias al tutorial de [midudev](https://github.com/midudev),
quien mostró cómo crear un portafolio minimalista en Astro, inspirado en el increíble trabajo de [BartoszJarocki](https://github.com/BartoszJarocki/cv).

El resultado: un sitio donde mi información personal, experiencia y formación se generan directamente a partir de un archivo JSON 📦.  
Así todo mi portafolio es **minimalista, fácil de mantener y súper rápido** ⚡.

---

## 🛠️ Tecnologías usadas
- 🚀 **Astro** – Para estructurar el sitio y lograr un build veloz.  
- 🟦 **TypeScript** – Tipado estático para mayor robustez.  
- 🎨 **CSS** – Estilos limpios y personalizados. 
- ⌨️ **Ninja Keys** – Para implementar atajos de teclado y mejorar la experiencia de usuario.   

---

## ✨ Características
- Portafolio generado desde un único archivo `cv.json`.  
- Diseño **minimalista y responsive** 📱💻.  
- Fácil de actualizar: solo edita el JSON y tu web se actualiza.  
- Inspirado en el modelo de **BartoszJarocki** y adaptado con mi propio estilo.  
- Schema del JSON del CV: [CV JSON Schema](https://jsonresume.org/schema/)
